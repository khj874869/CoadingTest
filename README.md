# CoadingTest
This is a auto push repository for Baekjoon Online Judge created with [BaekjoonHub](https://github.com/BaekjoonHub/BaekjoonHub).

## Repository quality

- Solutions are grouped by platform, difficulty, and problem number.
- Every Java and C++ solution is compiled independently in GitHub Actions, matching online-judge isolation.
- Run `python scripts/validate_solutions.py` locally before refactoring shared patterns.
